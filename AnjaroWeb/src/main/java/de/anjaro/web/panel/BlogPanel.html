Welcome to anjaro

Posted by Joachim Pasquali on February 12, 2013

This is the web application of anjaro, the java based roboter framework.
If you can read this, you should have been able to install the web application properly on your workstation. Congratulations!!

In the following blogs, I will tell you a little bit about anjaro, how to set it up, where you can find more detailed information and what you can do with this offline web application. We will start with the control panel, which will give you the possibility to control your anjaro robot and also to stream video and audio from your robot to the control panel.

At the end of this first blog, I just can say one thing: HAVE FUN!
